### CREATE VISUALIZATION ###

##Create Visualization
Grafana > Home > Dashboard > New > New dashboard > Add visualization > Select data source > Your data source > 

##Create Gauge
1. Create Query 
   ... > Your data source > Query > 

   A : Metrics browser = sum(kubelet_node_name{job="integrations/kubernetes/kubelet"})

2. Create Visualization
   ... > Your data source > Visualization > Gauge >

   Panel options 
   Title        : # of running k8s nodes 
   Value options 
   Show         : Calculate
   Calculation  : Last *
   Fields       : Numeric Fields
   Standard options 
   Unit         : Choose 
   Min          : 0 
   Max          : 4 
   Decimals     : auto 
   Display name : none 
   Color scheme : From thresholds (by value)
   No value     : -
   Thresholds 
   Red          : 80 
   Green        : Base

2. Create Visualization

##Create Logs
1. Create Query 
   ... > Your data source > Query > 

   A : {cluster="microbs", k8s_app="ecommerce"} |~ "critical|Critical|CRITICAL|error|Error|ERROR|fail|Fail|FAIL"

2. Create Visualization
   ... > Your data source > Visualization > Logs >

   Panel options 
   Title              : Logs pod kubernetes
   Logs         
   Time               : False 
   Unique labels      : False 
   Common labels      : False 
   Wrap lines         : False

   Enable log details : True
